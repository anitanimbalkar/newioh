Trackers

The Trackers opens an electronic Trackers Notes.

 

·To access a Trackers Notes, click the Trackers icon on the top right of the Dashboard Consultation.

 

 

·Select a new, blank tracker or open a tracker from those created earlier.

 

The Trackers allows the doctor to:

 

·Create and save electronic trackers for the patient being consulted.
·Click inside the Trackers to type the desired trackers for your patient. The Tracker allows you the following options:

 

To open a Tracker and edit, discard or save changes, and save tracker as template, follow the steps below:

 

§View all Trackers : Click the button. The Select Tracker drop-down menu displays with all trackers created and saved.
· Select the tracker to open. The selected tracker displays.

 

§Save Changes: Click the Save Changes button to save the changes made to the tracker.
§Discard Changes: Click the Discard Changes button to discard the changes made to the tracker.
§Save as Template: Click the button to save the tracker as a template for later use.

 

§Delete Tracker: Click the button to delete the tracker.

 

§Edit Tracker: Click the Edit button to edit the tracker.

 

To create a new tracker, follow the steps below:

 

 

§Click the New Tracker button.
§Click the Blank Tracker button.A new tracker displays.
§Create a tracker and follow the steps above to save as a template for recall later.
