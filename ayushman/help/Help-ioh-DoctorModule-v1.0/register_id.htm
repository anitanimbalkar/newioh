Register ID

 

Follow the steps below to register yourself:

 

·Click the RegisterMe button on the top right corner of the Home page of www.indianonlinehealth.com. The IndiaOnlineHealth - Registration Form opens.
·Click the drop-down arrow of the field Account Type. The drop-down menu opens showing four options. (* Mandatory field).
·Select the Doctor option (mandatory for registering yourself as a physician).
·Type your first name in the First Name field. (* Mandatory field).
·Type your middle name in the Middle Name field. (not mandatory).
·Type your last name in the Last Name field. (* Mandatory field).
·Type your complete email id (for example, xxxx@gmail.com) in the Email ID field. (* Mandatory field).
·Select your country of residence from the Country drop-down menu. Currently, only two countries, namely, India and Singapore are available on IndiaOnlineHealth application. (* Mandatory field).
·Type your telephone number in the Contact No. field. Note that the country code is displayed. The code cannot be changed. (* Mandatory field).
·Select and type your user name in the User Name field. (* Mandatory field). The user name is a unique name that you select for registering (one time) and logging into the application every time.

 

Password

 

·Select and type a password in the Password field. (* Mandatory field).
·Type the password you selected in the Confirm Password field. (* Mandatory field).

Security Question

 

·Select an option from the Security Question drop-down menu. Seven options are available for the security question. (* Mandatory field).
·Type your answer for the selected security question in the Security Answer field. (* Mandatory field). It is important that you select your security question and answer such that it is to easy recall later.
·Type the text&number string exactly as shown in the displayed image in the line below the Prove you're not a robot field. (* Mandatory field).
o IT IS IMPORTANT THAT YOU TYPE THE TEXT&NUMBER STRING EXACTLY AS SHOWN IN FIGURE TO ENSURE THAT YOU ARE AN AUTHENTIC USER AND IT IS NOT AN AUTOMATED FUNCTION.

 

oIF YOU HAVE ANY DIFFICULTY IDENTIFYING THE STRING, CLICK THE RELOAD BUTTON TO GENERATE A NEW STRING.
·Click the Terms&Conditions and read the terms&conditions.
·If you agree with the Terms&Conditions, click the small box on the field "I have read Terms & conditions completely and by checking the adjoining box I undertake to abide by them. I wish to register as a Consumer." A tick displays inside the box and OK displays in green.
·Click the button. The Account Activation screen opens.
