Current Prescription

The Current Prescription opens a Prescription form.

 

To access a prescription, click the Current Prescription icon on the top right of the Dashboard Consultation.

 

The Prescription allows the doctor to:

 

·Create and save an electronic prescription for the patient being consulted.
·Open a prescription saved as a template for the patient earlier/for any other patient saved earlier.
·Save a prescription created as a template for further recall and use.
