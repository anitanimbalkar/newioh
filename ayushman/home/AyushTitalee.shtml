• Tracker
    - Track Health Parameters
    - Compare parameter values with previous readings
    - Easy to access and view graphical view to monitor progress

  • Second consultation
    - Each Consultation is recorded by the doctor on the platform
    - Doctor & patient both can send a given consultation session for second opinion to the specialist, colleague or super specialist

  • Telemedicine
    - Video Consultation facility available in the application
    - If patient cannot access the application then a spike session with the doctor for simple problems can be done and the details can be then captured by the doctor on the platform against the patient.

  • Electronic medical record
    - Each Consultation is captured on the platform by doctor.
    - Each test report and corresponding parameter values are digitized on the platform
    - Digital prescriptions and reports become a part of the digital structure electronic medical record the customer.

  • Special attention to usability
    - Easy to use simple interface across all the functions including order placement, view EMR, Tracker, amongst others.
    - Single Page View across all the functions avoiding lot of redundant navigation.
